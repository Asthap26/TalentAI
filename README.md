# 🚀 TalentAI
### Intelligent Multi-Agent Resume Screening System

> From Resume Chaos to Smart Hiring Decisions in Seconds

---

## 🧠 Problem Statement

Recruitment teams process thousands of resumes daily across multiple formats such as PDF, DOCX, and plain text. These resumes often have inconsistent layouts and varying terminology for similar skills (e.g., "React.js" vs "ReactJS").

Traditional Applicant Tracking Systems (ATS) rely on keyword-based matching, which results in:
- Missing qualified candidates
- Poor matching accuracy
- Increased recruiter workload

This leads to inefficient hiring processes and suboptimal talent acquisition.

---

## 💡 Solution

TalentAI is a Multi-Agent AI system that:

- Parses resumes from multiple formats using intelligent extraction techniques
- Normalizes and maps skills to a structured taxonomy
- Performs semantic matching between candidate profiles and job descriptions
- Uses vector embeddings for accurate skill comparison
- Provides REST APIs for seamless integration with HR platforms

---

## 🚀 Key Innovation

- 🧠 Multi-Agent Architecture (Parsing + Normalization + Matching)
- 🔍 Semantic Matching using AI (not keyword-based)
- 🧩 Skill Taxonomy with intelligent mapping
- ⚡ Scalable API-first design
- 📄 Supports multiple resume formats (PDF, DOCX, TXT)
